# NayePankh-WebPage
NayePankh Foundation WebPage.

A webpage created for NayePankh Foundation as a part of internship project.

Developed by: Alyssa Suri

Project Overview:
This website was created for NayePankh Foundation using HTML, CSS, and JavaScript. The objective was to design a clean, user-friendly, and responsive webpage that effectively presents the foundation's mission, their story, and ways for visitors to get involved.

Features:
• Responsive design for different screen sizes
• Smooth scrolling navigation
• Home section with introductory content
• About Us section
• Our Stories section
• Certificates section
• Contact section
• Donate button for supporting the foundation
• Interactive hover effects and modern styling

Technologies Used:
• HTML5
• CSS3
• JavaScript

Folder Contents:
• index.html – Main webpage structure
• style.css – Website styling
• script.js – Interactive functionality
• images – Images used in the website
